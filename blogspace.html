BlogSpace

BlogSpace was my first encounter with APIs and learning how fetch works. I was able to use the API from Scrimba to create random blog posts. In the program, you are able to create a post with a title and a subject and when you click the post button, the inputs will generate into a post! A super simple program that can be turned into something like Twitter or Facebook. There are so many things I want to do to upgrade this program that I am still learning. I want to create a login section and a profile section. These will allow the user to create an account and be able to save their posts.

Project Purpose

I wanted to learn how APIs worked so I took a course on Scrimba that explained APIs. They provided a API for random blog posts and I thought it woud be fun to take my own spin on the app. I learned how to use fetch, what an API is, how to render posts and manipulate the DOM.
